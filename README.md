# Шаблон для университетских работ

## Cтруктура шаблона

Для лабораторных работ

```directory's tree
project
│   .gitignore
│   README.md
│   main.tex
│
└───images
    │
    └───image-subfolder
        │   ...
```

Для работ, требующих использование статей

```directory's tree
project
│   .gitignore
│   README.md
│   main.tex
│
├───articles
│   │   article.pdf
│   │   ...
│   
└───images
    │   
    └───image-subfolder
        │   ...
```
